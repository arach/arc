# @arach/arc-editor

Visual editor for creating Arc architecture diagrams.

## Usage

Run the editor locally:

```bash
npx @arach/arc-editor
```

This opens a browser with the Arc visual editor where you can:

- Drag and drop nodes to create architecture diagrams
- Connect nodes with styled connectors
- Choose from multiple themes and color palettes
- Export diagrams as TypeScript, JSON, SVG, or PNG

## Editor Features

- **Visual Design** - Drag-and-drop interface for placing nodes
- **Node Types** - Large, medium, and small node sizes
- **Connectors** - Straight and curved connection lines
- **Themes** - Default, warm, cool, and mono color palettes
- **Export** - Save as TypeScript config, JSON, or images

## Workflow

1. Run `npx @arach/arc-editor` to open the editor
2. Design your architecture diagram visually
3. Export the diagram config
4. Use `@arach/arc` to render the diagram in your React app
